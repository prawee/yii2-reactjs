# yii2-reactjs
This is React javascript library for Yii2

## Install 
Via Composer
``` bash 
$ composer require "prawee/yii2-reactjs:dev-master"
```

## Usage
On your view 
``` php
ReactJsAsset::register($this);
```
On your controller
``` php
ReactJsAsset::register($this->view);
```
